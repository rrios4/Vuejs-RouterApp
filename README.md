How to Run
----------------------------------------------------------
1. Run this command in client and root folder to install dependencies:
npm install

2. To start the application run:
npm run dev

3. Go to this link to see Vue page:
http://localhost:8080/

4. To view the JSON data go to:
http://localhost:5000/afcresults
